## Usage

```bash
./Install_CUDA.sh
```

or

```bash
python install_cuda.py
```

## Remove Only
Can remove `Nvidia-driver`, `CUDA`, `CUDNN`
```bash
./Install_CUDA.sh --uninstall
```

or

```bash
python install_cuda.py --uninstall
```
